Advanced Widgets for Designr themes. Fully compatible with other themes and SiteOrigin Page Builder

== Description ==
This plugin adds advanced, pre-designed widgets to your site that you can use in your theme. 

Available Widgets:
- Image Slider

Coming soon Widgets:
- Call to Action ( Various designs )
- Carousel
- Vertical split 
- Features Icons
- Pricing Tables
- Contact Info
- Team Members


== Installation ==
1. Download the plugin, then upload the zipped file to your site from the WordPress dashboard plugin uploader menu
2. OR - Upload the unpacked folder folder via FTP into /wp-content/plugins
